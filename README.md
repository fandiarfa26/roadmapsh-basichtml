# Basic HTML Website

A simple multi-page HTML website demonstrating semantic structure and SEO best practices.

## Pages

- **Home** - Welcome page with links to other sections
- **Projects** - Showcase of projects
- **Articles** - Blog-style articles
- **Contact** - Contact form and information
